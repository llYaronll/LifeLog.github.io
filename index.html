哪個更適合直播：Twitch 還是 YouTube？

2017年2月1日 BaBa

目前台灣人用的兩個平台不外乎就是Twitch跟Youtube兩個來實況，我覺得是個有特色，今天大概跟大家分享一下兩個平台的優劣到底在哪。

Twitch

優點
  1. 較有專業性，MOD管理員也可以直接裁決低能兒(以實況遊戲來說)
  2. 支援很智障的表情符號,例如BabyRage，OhMyDOG
  3. 支援很怪的BOT互動，可自行設定
  4. 與OBS的搭配設定較為簡便
缺點
  1. 客群較固定(意思就是Twitch使用群組比較死)
  2. 台灣網路有時候上傳量容易失智(造成丟格或者丟禎數的問題)
  3. 使用起來較死，廣泛性較低
  4. 想辦帳號的意願度較小，不像YT人人有一個，追蹤人數較難聚集
簡評

OBS使用起來確實比較簡單，如果是純玩遊戲我還是比較推薦Twitch，除了聊天室互動比較廣之外，BOT設定某方面可以省去很多實況主不必要的宣傳作業，大家也可以發現每個TWITCH實況版下面有些UP主會在下面設定很多有的沒的宣傳，也就是說這就是無形之中的「招牌」，經營久了客群會較固定，老觀眾也會知道你的經營風格是怎麼回事，但是TWITCH相繼的缺點就為明顯，會去辦TWITCH帳號的就是純粹看人玩遊戲，而只是稍微看的人不會特意去辦帳號，所以追蹤人數較難聚集。

Youtube

優點
  1. 客群非常廣，現在人人一個YT帳號
  2. 帳號與GOOGLE連結，帳號門檻低
  3. 使用者較多，人數好聚集，曝光性會高很多
缺點
  1. 實況串流功能較死，與OBS連結度較低
  2. 聊天室能玩出的花樣較少
  3. OBS聊天室每隔一段時間就要重新設定
  4. 訂閱人數較好聚集，相對的素質比較不一致

結論

直播在現今已經是一個趨勢，不論是FB或者YT都想搶這塊大餅，YT也懂這個環境的變化，所以他們推出了直播功能，YT開過的感想，不得不說非常的好聚集人潮，因為有訂閱你的人他只要開啟YOUTUBE，若你當下有再開實況就會出現在首頁，即時的互動性比TW高很多，也因為YT在這個世態下幾乎人人都有，不像TW較為古板，所以曝光率自然而然就高，不過你TW跟YT都開過就會很清楚，YT的功能就會比較少，而且我發現聊天室過一陣子就必須重新設定一次，這是比較麻煩的地方，而他及時的宣傳也會比較低，不像TW這樣拉下來是即時性的，所以每當開一次串流就必須設定一下你的主題跟宣傳。

Q&A

  1. Twitch跟Youtube要怎麼選擇

    2. 我覺得看使用者本身，YT雖然曝光率高但是訂閱你的人比較參差不齊，也就是說來的快去得也快，如果你是主攻單一主題的，那我建議你直接開YT不用想了，如果你想要甚麼都開，最後就會變成訂閱你的人沒有回流回來，也就是YT比較取像一個主題式的上傳，而不是把所有的東西都雜在一個YT頻道，久了你會發現訂閱數很高但是回流量怎麼這麼少．

  2. 臨時你是用哪一種?

    3. 兩個我都有經營，兩個平台我都用過，以我自己的話我是一個TWITCH跟兩個YT頻道，試過YT後我決定回去TW回去開，原因很簡單，我希望我的YT主題頻道比較偏向作品的上傳，而不是滿坑滿谷的實況紀錄，所以我在實況的時候我也會習慣性在把實況的紀錄放到另一個頻道，這就是所謂的區分，這也好讓你學習如何經營你的品牌，你的管理分類能力，YT使用過後是即時互動率真的很棒，但是我也不免擔心那些訂閱我TW的人被我放鴿子，所以最後我還是決定再TW開，之後把紀錄丟到副頻道的作法，這樣我的主頻道看起來也不會雜亂不堪。

  3. 怎麼雙修比較好

    4. 雙修還是看你的需求跟你在做的事情，像我就是做點評之餘開實況，所以我有必要把這兩個不同性質的東西區分開來，為了兩邊都達到平衡，宣傳就是很重要的手段，你要告訴你的觀眾你有一個YT頻道，同時也要告訴你的YT觀眾說你有一個ＴＷ頻道，區分管理固然很累，但是你會在無形之中學到一些經營的手段，久而久之你會明白怎麼做會比較好，而出來的成品也會鮮明許多，觀眾也會比較知道要往哪裡跑去哪裡看到他們要看的東西。
